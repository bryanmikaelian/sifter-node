Sifter-node

A NodeJs wrapper for the Sifter API written in CoffeeScript

Download as .zip Download as .tar.gz View on GitHub

Sifter - Node is an API wrapper for the Sifter API. It is written in Node and CoffeeScript. You can easily get it through npm. It has been verified to work against the latest version of Node.

Requirements

  • Sifter project for your company
  • Sifter API token
  • NodeJS vand CoffeeScript

Installing

npm install sifter-node
